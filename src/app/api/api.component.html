FootPrint API

Footprint API utilizes Spring Boot, Java, and MySQL. The API will require you to create a Footprint account to use the endpoints. And the data that will be accessible to you will be your data. Enjoy using API, and reach out if you have any questions. Here is the link for the Github Repository:FootPrint Backend

GET Request

The GET Request will allow you to retrieve your data for home, vehicle, user, and monthly storage data stored in the database. To GET your data, you must use the Basic Auth Token (your username and password) and user ID. You should visit the GitHub repository to understand the API better.

The Endpoints are:

  • /api/email to get your user account.
  • /api/footprint/(your id) will retrieve all of your data for vehicle, home, and user
  • /api/(your userID)/vehicle to GET your vehicle
  • /api/(your userID)/home to GET your home
  • /api/(your userID)/offsetters to GET your carbon offsetters
  • /api/(your userID)/image to GET your Profile Image
  • /api/(your userID)/storage to GET you monthly carbon storage

POST request

The POST Request will allow you to add your data for home, vehicle, and user stored in the database. To POST your data, you must use the Basic Auth Token (username and password) and user ID. You should visit the GitHub repository to understand the API better.

The Endpoints are:

  • /api/signup to create a user account.
  • /api/(your userID)/add/vehicle - to add your vehicle information to your account * A vehicle object is required.
  • /api/(your userID)/home - to add your home information * A home object is required.
  • /api/(your userID)/offsetters - to add your carbon offsetters * A offsetters object is required.
  • /api/(your userID)/upload to upload your Profile Image * A multipart file is required

DELETE Request

The DELETE Request will allow you to delete any home, vehicle, or user data stored in the database. To delete your data, you must use the Basic Auth Token (your username and password) and user ID. Please visit the GitHub repository to understand the API better.

The Endpoints are:

  • /api/(your userID) to delete user account
  • /api/(your userID)/vehicle to delete your vehicle * Vehicle Object must be sent with request
  • /api/(your userID)/delete/home to delete your home * Home Object must be sent with request
  • /api/(your userID)/offsetters to delete your carbon offsetters * Offsetter Object must be sent with request
  • /api/(your userID)/image to delete your profile Image
